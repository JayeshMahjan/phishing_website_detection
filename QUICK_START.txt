================================================================================
TROUBLESHOOTING
================================================================================

‚ùå Problem: "pip is not recognized"
   Solution: Make sure Python is installed and added to PATH
   Download: python.org

‚ùå Problem: "ModuleNotFoundError"
   Solution: Install requirements again:
   pip install -r requirements.txt --upgrade

‚ùå Problem: "FileNotFoundError: phishing_dataset.csv"
   Solution: Rename your CSV file to phishing_dataset.csv

‚ùå Problem: "Model file not found"
   Solution: Run train_model.py first before running app.py

‚ùå Problem: "Port 5000 already in use"
   Solution: Either:
   - Close other application using port 5000
   - Edit app.py and change port=5000 to port=5001

‚ùå Problem: Low accuracy (<80%)
   Solution: Check your dataset quality
   - Should have balanced classes (50/50 split)
   - Should have required columns
   - Should have clean data (no missing values)
